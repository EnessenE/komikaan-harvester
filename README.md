### The komikaan Harvester
This service is responsible for receiving requests from the "[filedetector](https://github.com/EnessenE/komikaan-file-detector)" to start downloading GTFS files and bulk importing them into the database.
When finished, it will submit all stops towards the "[gardeners](https://github.com/EnessenE/komikaan-gardener)"

Part of the overal komikaan.nl project, for more info see: [komikaan, the main repo](https://github.com/EnessenE/komikaan).
